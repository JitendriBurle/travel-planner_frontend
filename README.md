# Voyager - Plan Your Dream Journey

Voyager is your all-in-one travel companion for organizing itineraries, tracking expenses, and making every trip unforgettable.

## 🚀 Features

- **Itinerary Management**: Plan your days with activities, flights, and hotels.
- **Trip Documents**: Securely store and access your travel documents.
- **Expense Tracking**: Keep track of your spending in different currencies.
- **Smart Map View**: Get directions and search for locations directly on the map.
- **Trip Sharing**: Share your travel plans with friends and family.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, Lucide React, Leaflet.
- **Backend**: Express, Supabase (PostgreSQL & Storage), JWT Authentication.

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun

### Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```
3. Set up your `.env` files (see `.env.example`).
4. Run the development servers:
   ```bash
   # In frontend directory
   npm run dev
   # In backend directory
   npm start
   ```
